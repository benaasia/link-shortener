# linkshortener
Shorten links directly using [shorturl] shortcode

## Link Shortener Shortcode
Bạn có thể dễ dàng sử dụng shortcode để rút ngắn liên kết bằng plugin WordPress của chúng tôi. Bạn chỉ cần tải xuống bên dưới và tải nó lên trong WordPress và thế là xong. Không cần cấu hình nó vì nó đã được cấu hình sẵn cho bạn. Tất cả các liên kết của bạn sẽ được lưu trong tài khoản của bạn.

Không chia sẻ plugin này với bất kỳ ai mà bạn không tin tưởng vì họ sẽ có quyền truy cập vào toàn bộ API và về cơ bản là tài khoản của bạn.

## Hướng dẫn

Tải xuống plugin bên dưới
Chuyển đến quản trị viên WordPress > Plugins > Thêm mới > Tải lên plugin
Tải lên plugin có tên linkshortenershortcode.zip và kích hoạt nó. Plugin có tên là Link Shortener Shortcode.

Chi tiết plugin https://rutgon.com.vn/user/integrations/wordpress
